# 🔐 Smart Authentication System (JWT-Based)

## 📌 Description
A secure authentication system using JSON Web Tokens (JWT) for user login and authorization.

## 🚀 Features
- User Signup & Login
- JWT-based Authentication
- Protected Routes
- Secure API handling

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT
- REST APIs

## ⚡ How it Works
1. User logs in
2. JWT token is generated
3. Token is used to access protected routes
